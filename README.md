# VisKeys ⌨️

A Chrome Extension that improves the way a user learns keyboard shortcuts through generative AI suggestions, recognition over recall, and mapping suggestions to actions.

## Tech Stack 📚

- JavaScript
- HTML
- CSS
- Figma

## Using This Extension 🪜

1. Download VisKeys from Google Chrome Webstore (TBD).
2. Enable VisKeys within Extension Settings.
3. VisKeys will appear on web browser (toggle on/off).
4. Type prompt into the Generative AI Search Bar.
5. Tool then recommends a keyboard shortcut in the Suggestion Section. Simultaneously the keys to press based on the suggestion will show on the Virtual Keyboard.
6. Press these suggested keys or "Try Again".
7. Action of the keyboard shortcut command selection will execute as the user types the suggested keys.
8. Save the keyboard shortcut in the Keyboard Shortcut List if necessary, search for a new keyboard shortcut, or toggle the UI overlay off.
